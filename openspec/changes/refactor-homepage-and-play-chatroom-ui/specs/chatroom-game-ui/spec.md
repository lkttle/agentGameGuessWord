# Spec: Chat-Room Style Game UI

## Requirements

- Game room SHALL display players in a voice-chat-room style layout with large circular SecondMe avatars
- Each player's score SHALL be displayed below their avatar
- The game hint SHALL display pinyin initials (e.g., "CF") not Chinese characters
- A text input area SHALL be positioned at the bottom of the screen for player guesses
- Round log SHALL appear as chat-style messages in the center area
- The layout SHALL be responsive across mobile and desktop breakpoints
